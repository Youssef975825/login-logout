# 🔐 Login & Logout Page Design

A clean and modern User Interface (UI) design for a Login and Logout web page. This project was built as a practical exercise to master core web development skills, focusing on semantic layout structure and clean styling.

## ✨ Features

- Modern and minimalist UI/UX design.
- Fully responsive layout (compatible with different screen sizes).
- Clean, well-structured, and easy-to-read code.

## 🛠️ Tech Stack

- **HTML5:** For semantic web structuring.
- **CSS3:** For styling, layouts, and visual presentation.

## 📁 How to Run Globally

1. Clone or download this repository to your local machine.
2. Open the `index.html` file in any web browser (Chrome, Firefox, Edge, etc.) to view the project instantly.
